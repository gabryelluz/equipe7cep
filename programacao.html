Pesquisa sobre as principais TAGS do HTML, suas funcionalidades e exemplos de utilização.

HTML, que significa HyperText Markup Language, é uma linguagem de marcação usada para estruturar o conteúdo de uma página da web. As TAGs do HTML são elementos fundamentais para definir a estrutura e o significado do conteúdo em um documento HTML. Abaixo, estão listadas algumas das principais TAGs do HTML, suas funcionalidades e exemplos de utilização:

<html>: Essa TAG define o início e o fim de um documento HTML.
Exemplo: <html><head>...</head><body>...</body></html>

<head>: Essa TAG contém informações sobre o documento, como o título, estilos CSS e metadados.
Exemplo: <head><title>Título da Página</title></head>

<body>: Essa TAG define o corpo do documento, onde o conteúdo visível é colocado.
Exemplo: <body><h1>Título da Página</h1><p>Parágrafo de texto</p></body>

<h1> a <h6>: Essas TAGs definem os cabeçalhos do documento, sendo <h1> o mais importante e <h6> o menos importante.
Exemplo: <h1>Título Principal</h1>

<p>: Essa TAG define um parágrafo de texto.
Exemplo: <p>Isso é um parágrafo de texto.</p>

<a>: Essa TAG cria um hiperlink para outra página ou recurso.
Exemplo: <a href="https://www.exemplo.com">Link</a>

<img>: Essa TAG insere uma imagem na página.
Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">

<ul> e <li>: Essas TAGs são usadas para criar listas não ordenadas.
Exemplo: <ul><li>Item 1</li><li>Item 2</li></ul>

<ol> e <li>: Essas TAGs são usadas para criar listas ordenadas.
Exemplo: <ol><li>Item 1</li><li>Item 2</li>></ol>

<div>: Essa TAG é usada para agrupar elementos e criar blocos de conteúdo.
Exemplo: <div><p>Texto dentro de uma div</p></div>

<span>: Essa TAG é usada para aplicar estilos ou manipular parte do texto dentro de um elemento.
Exemplo: <p>Texto normal. <span style="color: red;">Texto vermelho.</span>></p>

<form>: Essa TAG cria um formulário para entrada de dados do usuário.
Exemplo: <form action="/submit" method="post">...</form>

: Essa TAG é usada para criar campos de entrada de dados em um formulário.
Exemplo: <input type="text" name="nome">

<textarea>: Essa TAG cria uma área de texto para entrada de múltiplas linhas em um formulário.
Exemplo: <textarea name="mensagem"></textarea>

<table>: Essa TAG é usada para criar uma tabela.
Exemplo: <table><tr><td>Célula 1</td><td>Célula 2</td></tr></table>

<tr>, <th> e <td>: Essas TAGs são usadas para criar linhas (<tr>) e células (<th> para cabeçalhos de tabela e <td> para células regulares) em uma tabela.
Exemplo: <table><tr><th>Cabeçalho 1</th><th>Cabeçalho 2</th></tr><tr><td>Dado 1</td><td>Dado 2</td></tr></table>

<br>: Essa TAG é usada para inserir uma quebra de linha.
Exemplo: <p>Linha 1<br>Linha 2</p>

<strong> e <em>: Essas TAGs são usadas para enfatizar o texto, sendo <strong> para ênfase forte (normalmente exibido em negrito) e <em> para ênfase leve (normalmente exibido em itálico).
Exemplo: <p>Este é um texto <strong>importante</strong> e este é um texto <em>ênfase</em>.</p>

<iframe>: Essa TAG é usada para incorporar um conteúdo externo, como outra página da web ou um vídeo.
Exemplo: <iframe src="https://www.youtube.com/embed/video_id" width="560" height="315"> </iframe>

<footer>: Essa TAG define o rodapé da página ou da seção.
Exemplo: <footer>© 2023 Exemplo.com. Todos os direitos reservados.</footer>

Essas são apenas algumas das TAGs mais comuns do HTML. Existem muitas outras TAGs disponíveis para diferentes propósitos e funcionalidades. É importante compreender a finalidade de cada TAG para estruturar corretamente o conteúdo de uma página da web e aproveitar ao máximo o HTML.
